Canvas Tote Bag

Role Design & Development
Team Kazi Ridwan, Nazmus Sakib
Company sitetrax

Client wanted a platform which has to be highly scaleable and needs to use their HoistCam Director & filestack api to store and receive image. Afterwards the images needs to be stitched according to users demand using openCV library and should be able to view the source images and also the resultant stitched image.

We built the frontend using ReactJS and used Redux for state management. The database was sql database and we used laravel as our backend engine for storing the site data. The images were stitched using the openCV library and used CUDA to accelerate the image manipulation process in linux server.erstellen.
